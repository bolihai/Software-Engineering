社团管理系统
- 功能
  - 登录，注册以及一些必要的有效性验证
  - 主页
    - 可以退出登录，加入社团，创建社团
    - 查看最新的活动，热门社团以及积极的成员
  - 活动
    - 所有活动可以筛选，搜索，分类以及加入社团
  - 个人
    - 查看个人所有的社团，参加的活动，发布的活动以及加入的社团
    - 查看他人的主页
